# Fonemi

## Inglese ("Received Pronunciation")

44 fonemi.

Vocali (18):

- Monottonghi (11):
  - i e.g. bean, many
  - u e.g. moon, you
  - u e.g. put
  - "schwa" e.g. sir, the
  - i chiusa
  - e e.g. hair (o square), met
  - a e.g. flood
  - oo e.g. door
  - o e.g. lot
  - aa e.g. path
  - a e.g. cat

- Dittonghi (7):
  - pie
  - how
  - choice
  - face
  - goat
  - theater, dear
  - cure

Consonanti (26):
- p/b, m
- f/v
- th/th
- t/d, s/z, n
- ch/j
- sh/s, r
- y
- k/g, ng
- h, "glottal stop"
- l chiara/velarizzata, w
